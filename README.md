# ClearCosmetics Website

A modern, mobile-first website for ClearCosmetics - a premium aesthetics brand with locations in Dublin and Marbella.

## Overview

This website features a clean, luxurious design with a focus on conversion and user experience. Built with HTML5, CSS3, and vanilla JavaScript, it's optimized for both desktop and mobile devices.

## Brand Identity

- **Primary Color**: Soft Nude/Blush (#f5e6e0)
- **Secondary Color**: Deep Charcoal (#222222)
- **Typography**: Playfair Display (headings) + Inter (body text)
- **Style**: Premium, minimalist, trustworthy

## Features

### ✨ Homepage Sections
1. **Hero Section** - Engaging introduction with clear CTAs
2. **About ClearCosmetics** - Trust-building content
3. **Treatments Preview** - Grid of available treatments
4. **Before & After Slider** - Interactive results showcase
5. **Testimonials Carousel** - Social proof
6. **Locations** - Dublin and Marbella clinic information
7. **Membership Teaser** - ClearCollective program preview
8. **Booking Section** - Direct links to Fresha booking
9. **Contact Form** - Lead capture

### 💉 Treatment Pages
Individual pages for each treatment with:

- Strong call-to-actions

## Customization Required

### 🔧 Essential Updates
1. **Contact Information**
   - Replace `[Dublin Phone]` with actual Dublin phone number
   - Replace `[Marbella Phone]` with actual Marbella phone number
   - Update `[YourWhatsAppNumber]` with WhatsApp contact
   - Add real clinic addresses

2. **Booking Integration**
   - Replace `[Your Fresha Dublin Link]` with actual Fresha booking URL
   - Replace `[Your Fresha Marbella Link]` with actual Fresha booking URL

3. **Images**
   - Replace all placeholder divs with professional images
   - Add before & after photos
   - Include clinic photos
   - Add professional headshots

4. **Social Media**
   - Update social media links in footer
   - Add actual Instagram/TikTok handles

### 📝 Content Customization
- Review and adjust all treatment descriptions
- Update pricing as needed
- Customize FAQ responses in chatbot
- Add more testimonials
- Adjust treatment areas/offerings
